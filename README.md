# vapes-resourcepack

Ресурспак, добавляющий модели вэйпов для плагина на вэйпы от OUTBREAK.

![preview](./preview.png)

## Получение и список моделей

```
/give @s carrot[item_model="outbreak_vapes:НАЗВАНИЕ_ПРЕДМЕТА"] 1
```
